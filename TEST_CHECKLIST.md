## 🔍 调试方法

### 打开浏览器控制台
按 F12 键,切换到 "Console" 标签

### 查看关键日志

#### 1. 检查登录状态
查找以下日志:
```
加载所有日程, currentOpenid: xxx
```
如果显示 `currentOpenid: null`,说明未登录

#### 2. 检查数据库查询结果
查找以下日志:
```
查询结果: {code: undefined, data: [...]}
```
- `code: undefined` 表示查询成功
- `data: [...]` 包含查询到的数据

#### 3. 查看错误信息
如果有错误,会显示红色的错误信息,例如:
```
loadToday error: Error: ...
```

## ❌ 如果还有问题

### 问题1: 还是提示"请先登录"
**原因**: 用户未登录或登录失败

**解决方法**:
1. 点击"匿名登录"按钮
2. 打开控制台,查看是否有错误信息
3. 检查CloudBase匿名登录是否已启用

### 问题2: 登录后还是加载数据失败
**原因**: 数据库安全规则未正确配置

**解决方法**:
1. 访问 https://console.cloud.tencent.com/tcb
2. 检查所有集合的安全规则是否已配置
3. 确保规则JSON语法正确

### 问题3: 数据库查询返回错误
**原因**: 安全规则中的字段名不匹配

**解决方法**:
1. 检查数据文档是否包含 `author_id` 字段
2. 检查安全规则中是否使用 `auth.openid` 或 `auth.uid`
